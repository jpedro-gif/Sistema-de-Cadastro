# Sistema de Cadastro

Sistema de cadastro de contatos no terminal feito em C#. Permite adicionar, listar e remover contatos, salvando os dados em arquivo de texto.

## Como usar

Abra o projeto no Visual Studio e pressione `Ctrl+F5` para rodar.

=== SISTEMA DE CADASTRO ===

Adicionar contato

Listar contatos

Remover contato

Sair

## Conceitos utilizados

- Classes e métodos
- Listas
- Loops (while, for)
- Condicionais (if/else)
- Leitura e escrita de arquivos
- Entrada do usuário pelo console
